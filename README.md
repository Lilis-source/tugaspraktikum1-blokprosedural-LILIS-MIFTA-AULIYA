## 📖 Deskripsi Tugas

Tugas ini bertujuan untuk memahami dan mengimplementasikan struktur blok prosedural dalam pemrograman basis data menggunakan MySQL. Program yang dibuat mensimulasikan validasi data akademik mahasiswa sebelum pengisian KRS.

---

## 🎯 Tujuan

* Memahami struktur dasar blok prosedural
* Menggunakan variabel dan konstanta
* Membuat logika validasi data akademik
* Menampilkan output program yang sistematis

---

## ⚙️ Fitur Program

Program ini memiliki beberapa bagian utama:

* Menampilkan identitas mahasiswa
* Validasi data akademik
* Kategori beban studi berdasarkan SKS
* Kategori performa akademik berdasarkan IPK
* Penentuan kelayakan pengambilan KRS
* Perbandingan dua mahasiswa

---

## 🗂️ Struktur File

* `program.sql` → berisi kode program utama
* `laporan.pdf` → laporan praktikum
* `README.md` → penjelasan tugas
* `dokumentasi/` → screenshot hasil eksekusi

---

## ▶️ Cara Menjalankan

1. Buka LARAGON dan jalankan MySQL
2. Masuk ke phpMyAdmin
3. Import atau copy file `program.sql`
4. Jalankan perintah:

   ```sql
   CALL program_utama();
   ```
5. Lihat hasil output

---

## 🧪 Skenario Uji

1. Data valid → mahasiswa layak mengambil KRS
2. Data tidak valid (pembayaran tidak lunas)
3. Data tidak valid (SKS atau semester tidak sesuai)

---

## 📊 Kesimpulan

Program berhasil mensimulasikan validasi data akademik mahasiswa menggunakan blok prosedural di MySQL. Program juga mampu memberikan informasi kelayakan KRS serta membandingkan performa akademik antar mahasiswa.
